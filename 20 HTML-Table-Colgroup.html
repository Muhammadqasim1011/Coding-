HTML Table Colgroup

The colgroup element is used to style specific columns of a table.
If you want to style the two first columns of a table, use the colgroup and col elements. The colgroup element should be used as a container for the column specifications.
Each group is specified with a col element. The span attribute specifies how many columns that get the style.
The style attribute specifies the style to give the columns.
Note: The colgroup tag must be a child of a table element and should be placed before any other table elements, like thead,tr, td etc., but after the caption element, if present.

Legal CSS Properties

There is only a very limited selection of CSS properties that are allowed to be used in the colgroup:
width property,
visibility property,
background properties,
border properties,
All other CSS properties will have no effect on your tables.
